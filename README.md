1. Description:

    -A script for Adobe After Effects that automatically creates realistic hair or long flexible element animations using Puppet Pins with procedural motion. The script includes an interactive GUI panel where you can adjust animation parameters and the number of pins, then apply them to a selected layer.

        -Each pin receives slightly different motion parameters, making the hair movement natural and organic. Pins are placed along a curve on the layer to simulate hair structure.

2. Features:

    -GUI panel for animation settings:

        -Frequency of wiggle (freq)

            -Wiggle amplitude (amp)

                -Decay of movement (decay)

                    -Number of pins (pins)

                        -Automatic Puppet Pins creation along the layer curve.

                            -Randomization of amplitude and frequency for each pin → more realistic motion.

                                -Allows recreating and updating the animation repeatedly without manually deleting pins.

                                    -Suitable for hair, ponytails, ribbons, vines, fabric, and any long flexible objects.

3. Requirements:

    -Adobe After Effects CC 2020 or later

        -Windows or macOS

        -Puppet Tool support (built-in AE feature)
